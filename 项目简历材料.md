# HazenixBlog 个人博客系统 - 简历材料

## 项目简介

HazenixBlog 是一个基于 Spring Boot + Vue 3 开发的现代化个人博客系统，支持文章发布、用户交互、内容管理等完整功能。系统采用前后端分离架构，实现了无状态身份认证、文章管理系统、评论互动、友链管理、树洞社区等核心功能，为个人博客提供了完整的解决方案。

---

## 技术架构

### 后端技术栈

- **核心框架**: Spring Boot 2.7+
- **数据持久化**: MyBatis + MySQL 8.0
- **缓存**: Redis (用户状态管理、JWT黑名单)
- **身份认证**: JWT (JSON Web Tokens) + ThreadLocal
- **连接池**: Druid
- **文件存储**: 阿里云 OSS
- **实时通信**: WebSocket
- **第三方登录**: Google OAuth、GitHub OAuth、微信登录
- **API文档**: Knife4j (Swagger)
- **构建工具**: Maven
- **容器化**: Docker + Docker Compose

### 前端技术栈

- **核心框架**: Vue 3 (Composition API)
- **构建工具**: Vite
- **路由管理**: Vue Router 4
- **状态管理**: Pinia
- **UI框架**: Element Plus
- **CSS框架**: Tailwind CSS
- **HTTP客户端**: Axios
- **富文本编辑**: Toast UI Editor / WangEditor
- **Markdown渲染**: markdown-it
- **工具库**: dayjs、highlight.js、js-cookie

## 项目亮点

### 1. 无状态身份认证机制

- ✅ **实现 JWT 无状态身份认证，Token 有效性基于拦截器集中校验**
- ✅ **使用 ThreadLocal 安全存储用户信息，确保业务层可直接获取身份数据**
- ✅ **解决 HTTP 请求无状态特性带来的身份认证问题**
- ✅ **支持 JWT 黑名单机制（基于 Redis），实现安全登出功能**

### 2. 前后端分离架构设计

- ✅ **采用 RESTful API 设计规范，前后端完全解耦**
- ✅ **前端使用 Axios 拦截器统一处理请求认证和响应错误**
- ✅ **后端实现统一异常处理和结果封装（Result/PageResult）**
- ✅ **支持跨域配置和统一的数据传输格式**

### 3. 数据持久化与缓存优化

- ✅ **使用 MyBatis 进行数据库操作，支持复杂 SQL 和动态查询**
- ✅ **集成 PageHelper 实现分页查询功能**
- ✅ **使用 Redis 缓存热点数据，提升系统性能**
- ✅ **数据库设计遵循第三范式，建立合理索引优化查询性能**

### 4. 功能模块化设计

- ✅ **后端采用 Maven 多模块架构（blog-server、blog-common、blog-pojo）**
- ✅ **前端组件化开发，按功能模块划分（article、admin、common、layout）**
- ✅ **统一的工具类封装（JwtUtil、AliOssUtil、HttpClientUtil）**
- ✅ **自定义注解和 AOP 切面实现功能增强**

### 5. 现代化前端体验

- ✅ **响应式布局设计，支持多终端适配**
- ✅ **集成富文本编辑器（Toast UI Editor），支持 Markdown 和 WYSIWYG 模式**
- ✅ **实现 Markdown 文章渲染，支持代码高亮、目录生成**
- ✅ **主题切换功能（支持亮色/暗色模式）**
- ✅ **使用 Pinia 进行全局状态管理，提升开发效率**

### 6. 安全与性能优化

- ✅ **密码使用哈希加密存储（不存储明文）**
- ✅ **JWT Token 校验拦截器，防止未授权访问**
- ✅ **使用 Druid 连接池优化数据库连接管理**
- ✅ **OSS 对象存储分离静态资源，减轻服务器压力**

### 7. 第三方集成能力

- ✅ **支持 Google OAuth、GitHub OAuth 第三方登录**
- ✅ **集成微信登录功能**
- ✅ **阿里云 OSS 文件上传和管理**

### 8. 实时交互功能

- ✅ **WebSocket 实时通信支持**
- ✅ **评论系统支持回复和点赞**
- ✅ **树洞社区功能（实时互动）**

## 数据库设计

系统包含以下核心数据表：

- **user**: 用户表（用户名、密码、邮箱、手机号、角色等）
- **article**: 文章表（标题、内容、封面、分类、标签、点赞、收藏、浏览数等）
- **comments**: 评论表（支持多级回复、点赞功能）
- **category**: 分类表
- **tags**: 标签表
- **article_tags**: 文章标签关联表
- **user_article**: 用户文章关联表（点赞、收藏关系）
- **links**: 友情链接表
- **tree_comments**: 树洞评论表
- **user_social_link**: 用户社交链接表
